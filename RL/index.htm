II. 最新信息

已知问题和解决方法

常规

 

・ 某些高速 USB 2.0 卡附带提供了未签名的主机控制器驱动程序。 如果您购买了 USB 2.0 插件卡并带有未签名的驱动程序，则可能碰到设备和 HP 软件之间的驱动程序安装问题或通信问题。 建议您始终对 USB 硬件使用 Microsoft WHQL 认证的主机控制器驱动程序。

・         以下网址中提供了即插即用打印/扫描驱动程序

软件包：

www.hp.com/support/ljm1210series

www.hp.com/support/ljm1130series

・         在仅满足最低系统要求的系统上，功能可能会受到限制。 要使用很多图像和视频，系统必须满足建议的系统需求以获得可接受的性能和可靠性。

・         设置和安装故障排除

如果无法识别您的设备：

  检查电缆是否符合 USB 2.0 高速标准。

  确保电缆连接正确。 拔出 USB 电缆，查看电缆末端，并查找与其形状匹配的 USB 端口。 确保电缆没有上下颠倒。 将 USB 电缆的一端完全插入到

HP 设备上的 USB 端口，并将另一端插入到计算机背面。

  检查外围设备电源是否打开。 如果电源关闭，则无法识别外围设备。

  确保 USB 电缆长度不超过 15 英尺或 5 米。

  检查是否启用了 USB 端口。 某些 PC 在出厂时禁用了 USB 端口。

  如果使用膝上型计算机的扩展坞站或端口复制器，请尝试将 USB 电缆连接到膝上型计算机的端口。

  某些 PC/膝上型计算机无法为 USB 端口充分供电。如果上述步骤仍未解决无法识别设备的问题，请尝试使用有源 USB 集线器。

・ 在泰语系统上，某些组件可能显示英文文本。

扫描

・         以较高分辨率扫描可能需要很长的扫描和处理时间，并且会生成很大的文件。 大多数情况下，默认分辨率可以提供兼顾质量和时间或文件大小的最佳效果。通过网络扫描也可能需要很长的时间。

・         扫描到文件时，HP LaserJet Professional 扫描软件的默认文件类型是 JPEG；而扫描到电子邮件时，默认文件类型是 PDF。 JPEG 文件支持 256 灰度（8 位）输出类型。 如果选择的输出类型为黑白（1 位），则生成的文件仍然是 JPEG 格式。

要获得真正的黑白输出类型，请通过设备的 HP LaserJet Professional 扫描软件的“设置”菜单访问“扫描图片设置”下面的“保存设置”对话框，然后在该对话框中将默认文件类型设置更改为 TIF 或 BMP。

・         确保将盖板正确放置在扫描仪的居中位置，这可防止杂光干扰扫描，从而获得最佳的自动剪切效果。 即使正确放置了盖板，一些图像的自动剪切效果仍可能不佳。 在这种情况下，需手动绘制剪切区域。

・         有关输入文件的 OCR 建议：

  300 dpi、8 位灰度或 24 位图像数据。

  亚洲语言（繁体中文、简体中文、日语和韩语）为 400 dpi、8 位灰度或 24 位图像数据。

  请勿将 1 位文件用于 OCR。

・         将一些纸张放在平板扫描的原件上有助于防止反面图像或泡沫垫映入扫描，还能使原件更平整。

・         如果扫描的原件弯曲、起皱或有划痕，则可能会导致在图像上出现阴影。

传真

・     根据电信线路质量的不同，按最高传真速度 (V.34) 传送或接收传真可能会失败。如果发生传送/接收故障，请在设备上的控制面板菜单中将传真速度更改为“中速”(V.17) 或“慢速”(V.29)，然后重试。仅模拟传真线路支持本产品。

如果使用的是 PBX、DSL 或 VOIP（基于 IP 的语音传输）数字传真线路，在出现传真问题时，请与数字线路提供商联系以了解兼容性问题。

III. 安装说明

在 Windows 环境中运行时，光盘设置或安装软件将检测正在运行的操作系统的版本和语言。

在 Windows XP、Windows Vista 32 位和 Windows 7 系统上安装软件

注： 要在 Windows XP、Windows Vista 或 Windows 7 操作系统上安装打印系统， 您必须具有管理员权限。

通过光盘安装打印系统

1.       在出现软件安装程序提示之前，不要将 USB 电缆插入到计算机上。

2.       关闭所有程序。

3.       插入光盘。

4.       如果 HP 打印系统安装程序在 30 秒后还没有启动，请完成步骤 5 至 7。

5.       从任务栏的“开始”菜单中选择“运行”。

6.       浏览到光驱的根目录。

7.       选择 HPSETUP.EXE 文件，单击“打开”，然后单击“确定”。

8.       按照提示进行操作。

卸载程序

在安装打印系统后，如果要删除所有 HP 打印系统组件，请使用 HP LaserJet Professional M1210 和 M1130 系列 MFP 程序组中的卸载图标。

启动卸载程序

1.       在“开始”菜单的 HP LaserJet Professional M1210 或 M1130 系列 MFP 系列程序组中，单击 HP LaserJet Professional M1210 和 M1130 系列 MFP 卸载图标。

2.       打印系统将指导您完成删除所有打印系统组件。

 

注：

如果进行了典型安装，并且未安装任何其它 HP 设备，则还需要执行下列步骤 1 和 2。 如果还安装了其它 HP 设备，则不要执行这些步骤，否则可能会破坏其它 HP 设备的某些功能，因为它们共享软件。

 

1.       打开 Microsoft Windows 控制面板。 双击“添加/删除程序”。

2.       您会看到“HP 扩展功能”和“HP 软件更新”的条目。

对每个条目，单击“更改/删除”，然后按照提示操作。
