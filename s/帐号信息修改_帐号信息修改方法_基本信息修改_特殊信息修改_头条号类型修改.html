帐号信息修改方法 #

头条号注册成功后可在“电脑端头条号后台 - 设置 - 帐号详情”中修改帐号信息，其中，基本信息也可以在“今日头条App - 我的 - 系统设置 - 编辑资料”中修改。修改后的信息，平台会进行审核，审核通过后方可正常使用。

修改次数限制

  • 一个自然月内，头条号的用户名可更改1次，头像和简介分别可更改5次；
  • 帐号类型每6个月修改一次。

基本信息修改 #

头条号的基本信息包括头像、帐号名称、帐号简介，可在电脑端头条号后台和今日头条App进行修改。其他帐号信息仅支持在电脑端修改。点击了解帐号资料的审核规范。

电脑端头条号后台：
在“电脑端头条号后台 - 设置 - 帐号详情”中，找到自己要修改的信息，点击修改即可。

今日头条App
在“今日头条App - 我的 - 系统设置 - 编辑资料”，可以修改头像、用户名、介绍及其他个人资料。

确认信息无误后，点击右上角【保存】，在出现【确认保存】弹窗时点击【确认】，即可保存修改。

如何申请更改帐号类型 #

申请修改帐号类型的流程：

  • 符合更改帐号类型的创作者可登录mp.toutiao.com，点击【设置】-【帐号详情】-【帐号信息】-【头条号类型】-【更改类型】，即可找到更改入口
  • 选择修改后的媒体类型，完善修改原因、组织名称、组织机构代码证/营业执照等信息；
  • 点击【提交】，平台将在3个工作日内进行审核；

💡小提示：

修改类型申请通过后6个月之内不可再次发起申请。
头条帐号类型说明详情请见：头条帐号有哪些类型。
申请修改帐号类型后，审核结果将在3个工作日内通过系统通知告知。

更改类型后认证&资质取消规则 #

由于认证和专业资质是根据帐号主体所拥有的资质材料或者创作能力申请的，修改类型时若实际发生了帐号主体的更改，相关的某些认证和专业资质将会失效，具体规则如下：

认证/资质取消规则：

  • 兴趣认证：个人和群媒体互转不失效，其他失效
  • 职业认证：企业和群媒体互转不失效，其他失效
  • 机构认证：其他组织转为新闻媒体不失效，其他失效
  • 健康资质和财经资质：机构之间互转不失效，其他失效（机构指的是所有主体非个人的帐号类型，包含企业、群媒体、新闻媒体、其他组织、国家机构）
  • 企业认证：企业和群媒体之间互转不影响企业认证，若从企业&群媒体改为个人，需要先在“头条号后台-右下角问题咨询”处联系客服，取消企业认证

帐号主体名称修改 #

目前仅支持企业、群媒体和新闻媒体帐号在注册成功后修改机构/公司名称。当帐号的主体发生改变或主体名称更改，可通过以下方式修改：

  • 在 “电脑端头条号后台 - 设置 - 帐号详情 - 机构/公司名称” 处，点击【更改主体】- 进入【更改帐号主体】页面。

  • 在对应输入栏填写与上传更改帐号主体所需的信息。

  • 点击【提交】，平台将在3个工作日内进行审核，申请通过后6个月内不允许再次修改；点击页面右上角【返回】，在此页填写的信息将不被保存。

运营人修改 #

「个人」类型帐号不支持修改运营人，除「个人」类型以外的机构类帐号，均可更改运营人信息。

  • 在 “电脑端头条号后台 - 设置 - 帐号详情 - 运营人” 处，点击【更改运营人】- 进入【更改运营人】页面；

  • 完善修改原因、运营人姓名、身份证号码等信息；

  • 确认信息无误后，请点击【提交】，提交更改运营人的申请；如要放弃修改，请点击页面右上角【返回】，回到帐号信息页，在此页填写的信息将不被保存。

申请通过后6个月之内不可再次发起申请。

网站修改 #

在注册时输入了机构网站的帐号，可以对网站信息进行修改。没有输入网站的帐号不支持后续添加网站信息。

  • 在 “电脑端头条号后台 - 设置 - 帐号详情 - 网站” 处，点击【更改网站】，进入网址修改页面；

  • 输入新的网站链接，勾选“ 我同意并遵守 《头条号用户注册协议》 ”，并点击下一步；

  • 根据提示进行网站管理权验证。

  • 完成验证，修改成功。

信息修改审核 #

帐号信息修改需要经过审核，审核通过后将显示修改过的内容。

  • 审核标准： 修改后的信息需要满足平台帐号准入标准，具体请参考帐号基本信息审核规范

  • 审核时间： 提交的修改信息一般会在1-2个工作日内审核，如遇周六日或者节假日会延期，辛苦耐心等待哦。

  • 审核结果： 审核完成后，帐号会收到系统消息通知，请注意查看审核结果。

常见问题 #

1.为什么帐号信息被重置？

若该帐号信息不符合平台规范会被重置，请参照 帐号基本信息审核规范。

2.企业/机构帐号主体注销了，如何修改帐号类型？

平台支持将帐号修改为主体法人的个人类帐号。

首先需要将帐号运营人修改为法人，再在“电脑端头条号后台 - 设置 - 帐号详情 - 头条号类型”处，点击更改类型。

如营业执照已因主体注销而被销毁，需要在上传【组织机构代码证/营业执照】处上传工商局出具的注销证明。

3.提现期间可否修改媒体类型？

不可以，需要等当前提现到账后才能修改媒体类型。

4.修改媒体类型为什么被永久拒绝？

如多次提供不符合规范的资料或提供虚假资料，将不支持在后台再次申请修改媒体类型。
